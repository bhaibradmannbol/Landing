# Add Collar Image

## Quick Steps

1. Save the collar image you showed me as: `public/images/collar.png`
2. The image will automatically appear in the Device Showcase section
3. It will rotate and scale as you scroll

## Image Specifications

- **Path**: `public/images/collar.png`
- **Format**: PNG (with transparent background preferred)
- **Size**: Recommended 1200x800px or larger
- **Aspect**: Landscape orientation works best

## What's Implemented

✅ **Apple-Style Navigation** - Fixed glassmorphic nav bar
✅ **Apple-Style Hero** - Full-screen with smooth animations
✅ **Device Showcase** - 3D rotating collar on scroll with GSAP
✅ **Glassmorphism** - Applied to Features, Solutions, Problems sections
✅ **Black Background** - Clean dark theme
✅ **Smooth Animations** - Framer Motion + GSAP ScrollTrigger

## Device Showcase Features

- Collar rotates on Y-axis (0° to -25°) as you scroll
- Scales up slightly (100% to 110%)
- Moves left with parallax effect
- Features fade in from right
- Blue glow effect behind device
- Glass cards for feature list
